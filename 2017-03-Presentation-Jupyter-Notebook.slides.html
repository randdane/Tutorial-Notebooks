Jupyter Notebook Introduction

History¶

IPython¶

  • Initial Release: 2001
  • "Command shell for interactive computing in multiple programming languages." Wikipedia.org
  • Interactive, parallelism, history, environment, etc...

Notebook¶

  • Interactive computing environment.
  • HTML / JavaScript with a kernel (Python (default), R, Julia, Haskell, 103 more...).
  • JSON (orderd list of input/output cells)
  • May contain live code, text, mathematics, plot, and rich media.
  • Built on IPython, MQ, Tornado, jQuery, Bootstrap, and MathJax.

Modes:¶

  • Command
  • Edit

Shortcuts¶

Navigation¶

Key(s) Action
H show shortcut menu
A add cell above
B add cell below
M markdown cell
Y code cell
DD delete cell
Shift + Enter run cell on move to next
Ctrl + Enter run cell on remain
Alt + Enter run cell and move to newly created cell

Editing¶

Key(s) Action
Enter go into insert mode
Escape exit insert mode
Tab code completion
Shift + Tab tooltips (3 versions)
Ctrl + S save

Markdown:¶

  • Headings (#)

  • Lists (*)

  • Blockquote

    This is an example.

  • Horizontal Rule (---)

  • Embedded code (three backticks) --> this text is inline code
